# Hanzi Brush 使用文档

## 基本信息

- App 名称：Hanzi Brush / 墨韵汉字 / 漢字ブラシ
- 版本：1.0.0
- 设备：iPhone
- 数据：全部保存在设备本地
- 账号：不需要账号
- 后端：不接后端，不主动发起网络请求

## 主要页面

### 今日

打开 App 后进入“今日”，可以看到今日汉字、拼音、含义和文化说明。点击“开始临摹”进入练习。完成并保存后，首页会显示今日已完成，连续天数和已练汉字数量也会更新。

### 练习

在“练习”页选择汉字，进入详情后可以查看部首、笔画、主题、文化说明和笔顺提示。点击“开始临摹”进入画布。画布支持书写、撤销、清空、显示/隐藏格线和对照字。没有笔迹时不能保存；写入后可以保存练习。

### 作品

在“作品”页选择汉字、模板和署名，点击“保存作品”。作品会出现在本地作品列表中，可以分享或删除。

### 字库

字库支持搜索汉字、拼音和含义，也可以按主题或收藏筛选。点击汉字卡片进入详情。收藏和练习状态会在返回后显示。

### 设置

设置页提供语言选择、隐私政策、支持页面、邮件支持和本地数据清除。设置页不显示 Bundle ID。

## 语言规则

用户显式选择 English、简体中文、日本語时，App 按选择显示。用户没有选择时，App 根据系统语言或地区推断，无法识别时使用英文。

## 隐私

练习记录、收藏、作品和语言选择保存在本地。App 只有在用户点击隐私政策、支持页面或邮件入口时，才会打开对应网页或邮件 App。
